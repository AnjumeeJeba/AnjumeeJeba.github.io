Thesis Paper

Implimentation of RSA Algorithm and Digital Signature scheme to strengthen data surveillance of smart office with blockchain in fog computing architecture
Authors : Anika Tabassum, Humayra Anjumee Jeba, Tasnim Kabir Mahi, S. M. Salim Rez

The implementation of cloud computing has been on the rise in the office management system. However, due to some concerning issues regarding provided security by the cloud, it has been losing its publicity. Moreover, the complete reliance of cloud computing over the internet is also acting as a drawback towards the usage of this system. With the introduction of fog computing with its advantageous feature of decentralization, this system has been replacing traditional cloud systems in various organizations. In this paper, we propose a smart office system which is based on fog computing, a distributed architecture where users, storage, information sharing are supervised by many local access points. Blockchain is implemented to ensure overall system security and user identity has been conducted by the creation of block in the system. All blocks are encrypted with special cryptographic key which makes data manipulation by any intruder without the legit owner quite impossible. The overall transaction process was performed using the common bitcoin framework, which is carried out in a peer-to-peer network environment without the influence of any dominant authority. Information sharing has been implemented with the help of RSA algorithm and digital signature scheme. Decentralized fog storage system has been adopted for storage, ensuring the protection of the stored data through cryptographic hash encryption and data replication to minimize loss of data.
